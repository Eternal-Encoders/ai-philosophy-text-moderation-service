# AI-сервис фильтрации текстов

**Описание:**  
Сервис предназначен для проверки текста на соответствие теме философии и на токсичность. Используется для обеспечения корректности вводимых сообщений в интерактивном AI-симуляторе философских дилемм.  

Установить зависимости:
```
pip install -r requirements.txt
```

Запустить:
```
uvicorn app.main:app --reload
```

Запуск тестов:
```
pytest
```

Проверка на соблюдение PEP 8:
```
flake8 app/ tests/
```
